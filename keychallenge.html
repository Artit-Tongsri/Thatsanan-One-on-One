Key Challenge

"ความท้าทายหลัก" หรืออุปสรรคสำคัญในการทำงาน

1.การพัฒนาทักษะความรู้ : เป็นโปรเจ็คใหม่ที่ไม่คุ้นเคย ฝั่ง Dtac และ True ทำงานไม่เหมือนกัน เท่ากับโปรเจ็คนี้เราจะได้เจอปัญหามากขึ้น ได้ความรู้กับสิ่งใหม่ๆมากขึ้น

2.การพัฒนาด้านการสื่อสาร : ต้องเป็นหลีดในการ UAT กับ User ทำให้เราได้ฝึกการพูดการสื่อสารยังไงให้คนเข้าใจ รวมไปถึงการสื่อสารกับคนในทีมเวลาติดปัญหาต่างๆต้องรู้ว่าต้องขอความช่วยเหลือจากใคร ซึ่งก็จะมีหลายคนมาก

3.ด้านแรงกดดัน : จากปริมาณงานที่ค่อนข้างเยอะ และถือหลายโปรเจ็คหลายงานในการนำขึ้น1รอบ ต้องอดทนกับแรงกดดันและความพยายามอดทนทำงานให้เสร็จตามกำหนด

4.ด้านการบริหารจัดการ : ต้องจัดการลำดับความสำคัญของงาน และบริหารเวลาตลอดเวลา

5.ด้านร่างกายและจิตใจ : ด้วยปริมาณงานที่ค่อนข้างเยอะ ทำงานนอกเวลาบ่อยมาก จนพักผ่อนน้อย ป่วยบ่อยและท้อใจเป็นบางครั้ง ต้องพยายามหาแรงบัลดาลใจและฮิลใจตัวเองเสมอ
