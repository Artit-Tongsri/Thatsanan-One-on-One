Tool

เครื่องมือ หรือ อุปกรณ์ ที่ใช้ในการทำงานให้สำเร็จลุล่วง

Tool รายละเอียดการใช้งาน
Software
Microsoft Team ใช้สำหรับการสื่อสาร (แชท, โทร), การประชุมออนไลน์ (วิดีโอคอล, แชร์หน้าจอ, บันทึกได้), การจัดการไฟล์ (แชร์และแก้ไขเอกสารร่วมกัน), และการจัดระเบียบงานสำหรับทีมและองค์กร
Jira ใช้เพื่อ บริหารจัดการโครงการและติดตามการทำงาน (Project Management & Issue Tracking)
Jenkins ใช้เพื่อตรวจสอบสถานะการเขียนโค้ดและการ deploy
Gitlab ใช้เปิด-ปิด Toggle เพื่อทดสอบ และใช้ปรับโค้ด กรณีต้องการทดสอบ Time Out
Mongo Robo3t ใช้สำหรับเชื่อมต่อ ดู แก้ไข และจัดการข้อมูลในฐานข้อมูล
Kibana ใช้ตรวจสอบการทำงานของระบบ,Debug โค้ด, ตรวจสอบข้อผิดพลาด (errors),Log
OMX Admin ใช้ตรวจสอบการทำงานของระบบ, ตรวจสอบข้อผิดพลาด (errors),Log ,ตรวจสอบ Status Order
CRM monitor ใช้ตรวจสอบการทำงานของระบบ, ตรวจสอบข้อผิดพลาด (errors),Log ,ตรวจสอบ Status Order ,เอกสารแนบ
ECM ใช้ตรวจสอบการทำงานการส่งค่าของระบบ,เอกสารแนบ,Eform
CCB ใช้สร้าง User เพื่อเข้าใช้งานระบบ ,ตั้งค่าสิทธการใช้งาน ,ตั้งค่า Screen Code ,ตรวจสอบรายละเอียดเบอร์โทรศัพท์
Smart Connect Manager ใช้เพื่อตรวจสอบหมายเลขเครื่องชำระเงิน(True)
Registry Editor ใช้เพื่อตั้งค่าหมายเลขเครื่องชำระเงิน(True)
Figma ใช้เพื่อดูการออกแบบ UX/UI และFlow ของระบบ
CRM Lite , CRM Plus ใช้เพื่อตรวจสอบข้อมูลฝั่ง True
Customer Portal ใช้เพื่อตรวจสอบข้อมูลฝั่ง Dtac
CRM Cbel ใช้เพื่อตรวจสอบข้อมูลฝั่ง True
One App ใช้เพื่อทดสอบการทำรายการผ่าน One App , การยืนยันตัวตน
Hardware (Skills)
Card Reader ใช้ทดสอบการอ่านบัตรประชาชน การยืนยันตัวตนจากการอ่านบัตร
Passport Scanner ใช้ทดสอบการScan Passport การยืนยันตัวตนจากการScan Passport
Webcam ใช้ทดสอบการยืนยันตัวตนจาก Webcam
Scanner barcode ใช้ทดสอบการสแกนบาร์โค้ดหทายเลขซิม
